# 📝 TZ 101Hotels — Список комментариев

## 📌 Описание

Приложение реализует страницу со списком комментариев с возможностью:

* добавления комментариев
* удаления комментариев (без перезагрузки страницы)
* сортировки
* постраничного просмотра

---

## ⚙️ Стек технологий

* **PHP 7.4**
* **CodeIgniter 4**
* **MySQL 8**
* **jQuery 3**
* **Bootstrap 4**
* **Docker / Docker Compose**

---

## 🚀 Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone <repo_url>
cd TZ_101hotels_crud
```

### 2. Запустить контейнеры

```bash
docker compose up --build -d
```

### 3. Установить зависимости

```bash
docker exec -it tz_hotels_web bash
composer install
exit
```

### 4. Выполнить миграции

```bash
docker exec -it tz_hotels_web php spark migrate
```

### 5. Открыть в браузере

```text
http://localhost:8080
```

---

## 🧩 Функциональность

### ✅ Комментарии

* добавление комментариев (AJAX, без перезагрузки)
* удаление комментариев (AJAX)
* отображение списка

### ✅ Пагинация

* по **3 комментария на страницу**
* переход между страницами

### ✅ Сортировка

* по `id`
* по дате добавления (`created_at`)
* направления:

    * по возрастанию
    * по убыванию

### ✅ Валидация

* email валидируется:

    * на клиенте
    * на сервере

---

## 📅 Работа с датами

В модели используются **два разных поля**:

### `date`

* строковое поле
* вводится пользователем
* формат: `ДД.ММ.ГГГГ ЧЧ:ММ`
* соответствует ТЗ: *"дата выбирается создателем"*

### `created_at`

* служебное поле
* заполняется автоматически
* используется для:

    * сортировки
    * отображения даты добавления

---

## 📌 Важно

Поле `name` в базе данных используется для хранения **email пользователя**.

Это сделано в соответствии с формулировкой ТЗ:

> `name` — почта создателя

## 🔌 API (используется через AJAX)

| Метод  | URL              | Описание             |
| ------ | ---------------- | -------------------- |
| GET    | `/`              | список комментариев  |
